## 🧠 Core Principles

My craft is defined by a set of core beliefs and passions.

> **Coding Power Free:** I build tools that unlock potential and automate the mundane. I believe code should be liberating, not restrictive.
>
> **Terminal Addicted:** The command line is my home. Efficiency, speed, and control are found in the terminal.
>
> **Vibe Code:** I write code that feels right. Clean, expressive, and with an innate sense of flow. It's about the rhythm as much as the logic.
>
> **Open Source Lover:** I stand on the shoulders of giants and contribute back. The future is built in the open.
